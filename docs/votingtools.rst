======================
Voting Tools
======================

    🥇 1. Helios Voting (BEST for security + verifiability)
    ===============================================

    Helios Voting

    Why it’s great
    --------------
    - 100% free and open-source
    - Strong cryptographic verifiability (end-to-end proof of votes)
    - Supports linking/uploading documents like proposals or PDFs
    - Each voter gets a tracking code for audit verification
    - Used in universities, student governments, NGOs

    Limitations
    -----------
    - Not very “plug-and-play”
    - Basic UI compared to commercial tools

    Best choice
    -----------
    👉 Best choice if you care most about trust, transparency, and auditability


    🥈 2. OpaVote (best simple free tier + ranked voting)
    ====================================================

    OpaVote

    Why it’s good
    -------------
    - Free open-source version available
    - Supports ranked-choice, approval voting, surveys
    - You can attach or reference documents in ballots
    - Very fast setup (good UX for admins)
    - Secure authentication options

    Limitations
    -----------
    - Advanced features may require paid plan
    - Not as strong as Helios in cryptographic transparency

    Best for
    --------
    👉 Best for clubs, student elections, small organizations


    🥉 3. Google Forms (surprisingly practical option)
    =================================================

    Google Forms

    Why people still use it
    -----------------------
    - Completely free
    - Easy document embedding (Google Drive PDFs, proposals, links)
    - Works well for small internal votes
    - Automatic result charts + exports

    Limitations
    -----------
    - Weak election security (no strong anti-voting manipulation)
    - No real audit trail / anonymity guarantees

    Best for
    --------
    👉 Best for non-official, low-risk voting (school, teams, polls)


    🧾 4. ElectionBuddy (limited free tier but very reliable)
    =========================================================

    ElectionBuddy

    Why it’s used widely
    --------------------
    - Professional-grade voting system
    - Supports motions, bylaws, and document-based voting
    - Strong audit logs + encrypted ballots
    - Designed for formal organizational elections

    Free aspect
    -----------
    - Free for very small elections / trial use only (up to ~20 voters in some cases)

    Best for
    --------
    👉 Best for serious organizational elections where documents matter


    🥈 5. NemoVote (free demo + strong GDPR compliance)
    ===================================================

    NemoVote

    Why it’s good
    -------------
    - Handles motions, PDFs, proposals, attachments
    - Strong compliance (GDPR-focused)
    - Very easy for voters
    - Free demo / small test elections

    Limitations
    -----------
    - Full version is paid for larger elections

    Best for
    --------
    👉 Best for EU-based organizations needing compliance + document handling
